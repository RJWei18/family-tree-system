# 系統測試驗收文件 (System Test Acceptance)

## 1. 測試環境
-   **Browser**: Chrome (Latest), Safari (iOS Latest)
-   **Display**: Desktop (1920x1080), Mobile (iPhone 12/13/14 format)

## 2. 驗收標準 (Acceptance Criteria)
1.  **零嚴重錯誤 (Zero Critical Bugs)**：無白畫面、無崩潰、無資料遺失。
2.  **功能完整性**：所有核心功能 (CRUD、家族樹顯示、搜尋) 皆可正常運作。
3.  **UI/UX 一致性**：所有頁面風格統一，且在手機版無破版。
4.  **照片載入**：Google Drive 照片需能正常顯示，無 403 錯誤。
