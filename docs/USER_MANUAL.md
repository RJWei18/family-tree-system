# 系統操作手冊 (User Manual)

## 1. 成員管理 (Member Management)

### 1.1 新增成員
1.  進入首頁「成員管理」。
2.  點擊右上角 **「+ 新增成員」** 按鈕。
3.  填寫基本資料：
    *   **姓名**：必填。
    *   **性別**：影響家族樹節點顏色 (藍/粉)。
    *   **出生/過世日期**：支援 `YYYY-MM-DD` 格式。若已過世，請勾選「Status: 歿」或填寫過世日期，頭像將自動轉為黑白。
    *   **照片網址**：請參考下方「照片管理」章節。

### 1.2 建立關係
您可以透過兩種方式建立關係：
1.  **父母關係**：在成員編輯視窗中，直接選擇「父親」與「母親」。
2.  **配偶關係**：在成員列表點選該成員，進入編輯模式，在「配偶」欄位選擇對象。

### 1.3 照片管理 (Google Drive 自動化)
本系統支援直接讀取 Google Drive 照片。
1.  將照片上傳至 Google Drive 指定資料夾，檔名需包含成員名字 (如 `魏錕源.jpg`)。
2.  開啟 Google Sheet 資料庫。
3.  點選上方選單 **「家族樹工具」** -> **「更新照片連結」**。
4.  系統會自動填入連結，回到網頁重新整理即可看到頭像。

## 2. 家族樹操作 (Family Tree Navigation)

### 2.1 瀏覽操作
-   **縮放 (Zoom)**：使用滑鼠滾輪，或點擊右下角的 `+` / `-` 按鈕。
-   **移動 (Pan)**：按住滑鼠左鍵拖曳畫布。
-   **適應視窗 (Fit View)**：點擊右下角的「四向箭頭」圖示，自動將整棵樹縮放至可視範圍。

### 2.2 搜尋成員
1.  點擊左上角的 **搜尋圖示**。
2.  輸入姓名關鍵字。
3.  點選列表中的成員，家族樹會自動飛越並聚焦到該成員節點。

### 2.3 匯出圖片
-   點擊右上角的 **「下載圖片」** 按鈕，系統將當前家族樹存為 PNG 圖檔。

## 3. 壽星月曆 (Birthday Calendar)

### 3.1 查看壽星
-   進入「壽星月曆」頁面。
-   **左側列表**：顯示本月所有壽星及其年齡。
-   **右側日曆**：以月曆形式呈現，點擊日期可查看當日壽星。

### 3.2 切換月份
-   點擊日曆上方的 `<` 或 `>` 按鈕可切換月份。
-   點擊 **"Today"** 可快速回到當前月份。
